## Pune 7/12 Selenium Scraper

### Features
- CSV-based input (wards & parcels)
- CAPTCHA pause for manual solving
- Resume support (skips completed)
- Retry support via logs
- ZIP output of images

### How to Run
1. Clone the GitHub repository
2. Install dependencies using "pip install -r requirements.txt"
3. Update input.csv with sample ward and parcel numbers
4. Run the craper using "python scraper.py"
5. Review the downloaded images and generated ZIP file in the output folder
